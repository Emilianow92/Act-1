El programa funciona mediante una interfaz de línea de texto en la consola. Al iniciarse, solicita el nombre del cliente y muestra un catálogo de vehículos con sus precios y características. El usuario puede seleccionar uno o más autos, y el sistema va sumando automáticamente el costo de cada selección.

Al finalizar la compra, el programa genera un ticket de venta que incluye el nombre del cliente, la fecha de la transacción y el total a pagar. De esta manera, se simula de forma sencilla el proceso de venta de vehículos utilizando Java.
